# MengenLehre

## ⊂

⊂ = Teilmenge

### Beispiel

- Jedes Element das in E ist muss auch in C sein
- E ⊂ C
- E={z,r,u}, C={r,d,g,t}
  Ergebniss: Falsch

## ∩

∩ = Schnittmenge

### Beispiel

- Jedes Element was in A und B ist
- A ∩ B
- A={1;2;3;4;5}, B={2;5}
  Ergebniss = 2, 5

## ∪

∪ = Vereinigungsmenge

### Beispiel

- Jedes Element was in A oder B ist
- A ∪ C
- C={3;5;7;9}, A={1;2;3;4;5}
- Ergebniss = 3, 5, 7, 9, 1, 2, 4,

## c

c = Ergänzungsmenge

### Beispiel

- Jedes Element was nicht in Bc ist aber in der Grundmenge.
- also wenn grundmenge={1;2;3} B={2, 5, 6, 7}
- Ergebniss = 1, 3

## /

- / ist wie minus
- Also wenn C={3;5;7;9}, A={1;2;3;4;5}
- C/A
  Ergebniss = 7, 9

# Joins

## Inner Join

Inner Join ist wie ∩

## left join

Links: Alle Zeilen bleiben zu 100 % erhalten. Niemand wird gelöscht.

Rechts: Es werden nur die Teile behalten, die zur linken Tabelle passen. Was rechts keinen Partner hat, wird ignoriert.

- links = zuerts gennante

## Right join

## FULL OUTER JOIN
